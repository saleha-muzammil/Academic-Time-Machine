## Purpose
This repository serves as a comprehensive resource for the **Software Construction and Development (SCD)** course. It contains practical implementations, personal practice codes, and demonstrations of various advanced Java topics, ranging from GUI development to network programming and software architectures.

## Topics Covered
### Core & GUI Development
- **JavaFX** (`javaFx/`): Fundamentals, Panes, Layout Management, FXML integration.
- **Java Swing** (`_graphics/`, `layouts/`, `practice/`): 
  - Custom Graphics (JStar, DrawImage).
  - Layout Managers (BorderLayout, BoxLayout, FlowLayout).
  - GUI Components, Event Listeners, and Tables.

### Advanced Java Concepts
- **Multithreading** (`Thread/`): Thread management, Synchronization, Producer-Consumer Pattern.
- **Database Connectivity**:
  - **JDBC** (`jdbc/`): Java Database Connectivity.
  - **ORM/Hibernate** (`ORM/`): Object-Relational Mapping.
- **Network Programming**:
  - **Sockets** (`sockets/`): Echo Server, Chat Systems, Games (Number Guessing, Rock Paper Scissors), Monitoring Systems (O2, Attendance).
  - **RMI** (`RMI/`): Remote Method Invocation.
  - **Web Connectivity** (`Web_Connectivity/`): Handling HTTP GET and POST requests.

### Software Engineering & Architecture
- **Architectures**:
  - **MVC Pattern** (`mvcArchitecture/`, `practice/`): Calculator, Temperature Converter.
  - **Layered Architecture** (`layerdArchitecture/`): Separation of GUI, Service, and DAO layers.
- **Serialization** (`serielization/`): Object persistence.
- **Reflection** (`reflection/`): Runtime logic inspection.
- **Annotations** (`annotation/`): Custom annotation processing.
- **Unit Testing** (`testing/`): JUnit test cases.
